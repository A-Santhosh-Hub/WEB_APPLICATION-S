# **SanKeep — Smart Notes**

SanKeep is a fast, beautifully designed, and production-ready Progressive Web Application (PWA) for taking and organizing your notes. Inspired by modern design languages like Android's Material You and OriginOS, it offers a seamless, app-like experience directly in your browser.

## **✨ Features**

* **Rich Note-Taking:** Create detailed text notes instantly.  
* **Drawing Canvas:** Sketch your ideas and save them directly with the built-in drawing editor.  
* **Image Attachments:** Upload and preview images directly inside your note cards.  
* **Organization:** Sort your notes using custom Folders, Pin your most important notes, and manage the rest in Archive or Trash.  
* **Customization:** Color-code individual notes and apply custom aesthetic wallpapers to your workspace.  
* **Multi-Account Support:** Create and seamlessly switch between multiple user profiles.  
* **Export Anywhere:** Download your notes locally as .txt, .doc, .html, or .pdf files.  
* **Real-time Search:** Instantly filter through your notes to find exactly what you need.  
* **Dark & Light Mode:** Fully adaptive theming that you can toggle at any time.  
* **Offline Capable:** Works offline and is fully installable as a Progressive Web App (PWA) on mobile and desktop.

## **🚀 How to Use**

Simply open the SanKeep.html file in any modern web browser to start using the application instantly. No complex setup or installation required\!
